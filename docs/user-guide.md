Set up a configuration file to make it just
right for your preferences.

## Loading Data

To learn about the supported formats and features concerning loading
a dataset with Datui, see the [Loading Data Guide](user-guide/loading-data.md).

## Querying, Filtering, and Sorting

### Query Syntax

Datui's query syntax makes filtering, grouping, and aggregating your
dataset easy. You can also use it to make drilling down into subgroups of your
data simple.

To learn more about it, see the [Querying Data Guide](user-guide/querying-data.md).

### Filtering and Sorting Dialogs

In addition to a query language, filtering and sorting can be done by using
a dialog window in the TUI. See [Filtering and Sorting](user-guide/filtering-sorting.md) for more.

## Pivot and Melt

See the [Pivot and Melt Guide](user-guide/reshaping.md) to learn about reshaping your data.

## Save your Work with Templates

Save and reapply your queries, sorts, and transformations using the [Templates](user-guide/templates.md)
system. Templates save your customizations so they can be quickly re-applied later to the same
or similar files.

## Analysis Features

Use tools to explore characteristics or extract insight from your data. Datui contains
a number of tools, such as:

- Discover relationships with a correlation matrix
- Learn about the characteristics of your data with distribution analysis

See the [Analysis Features Guide](user-guide/analysis-features.md) for full details.
