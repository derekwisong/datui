Datui is a terminal application for exploring and analyzing data.

Full documentation: https://derekwisong.github.io/datui/

Use Datui to:

- Read tabular data from many formats
- Work efficiently with large datasets (hive partitioning, lazy evaluation)
- Export data to file
- Create charts in the terminal and export them to a file
- Save your filter, sort, and query to a template for reuse
- See relationships with a correlation matrix
- And more
